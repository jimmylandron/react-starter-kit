# React Starter Kit
This is a React Webpack boilerplate. 


### Installing

Install dependencies

```
npm install
```


Start the server

```
npm start    Served on localhost:8080
```

Build for production

```
npm run build
```


## Built With

* [React js](https://reactjs.org/)
* [Webpack](https://webpack.js.org/)
